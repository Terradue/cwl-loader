# Tutorials

Tutorials are guided lessons for learning CWL Loader by doing something small
and complete.

Start with [First steps with CWL Loader](first-steps.md). It creates a tiny CWL
document, loads it with the Python API, and writes a normalized YAML file.

The [notebook examples](../examples.ipynb) provide a longer interactive
walkthrough that loads workflows, serializes CWL, and parses existing Python
data structures.
